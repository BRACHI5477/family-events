# יומן אירועים משפחתי 👨‍👩‍👧‍👦

מערכת ווב מודולרית לניהול משפחה, אירועים ותזכורות אוטומטיות — עם תמיכה מלאה בתאריכים **עבריים ולועזיים**, לוח שנה דו-לוחי, חישוב גיל אוטומטי, ושליחת מיילים מעוצבים.

נבנתה לפי האפיון ב-[Family.md](Family.md).

## הרצה מהירה

דרישה: **Node.js 22 ומעלה** (המערכת משתמשת ב-SQLite המובנה של Node — ללא צורך בהתקנת מסד נתונים).

```bash
npm install
npm start
```

לאחר מכן פתחו בדפדפן: **http://localhost:3000**

**כניסת דמו:** `admin` / `1234`

## מה כלול

| מודול | תיאור |
|-------|-------|
| 🏠 מסך הבית | לוח שנה חודשי/שבועי/שנתי (עברי + לועזי), אירועי היום/שבוע/חודש, חיפוש מהיר |
| 👥 בני משפחה | ניהול מלא: הוספה, עריכה, מחיקה, ארכוב + חישוב תאריך עברי אוטומטי |
| 🎉 אירועים | ימי הולדת, נישואין ואירועים מותאמים — חישוב לפי עברי/לועזי/משולב |
| 🔔 תזכורות | כללי תזכורת (חודש/שבועיים/שבוע/3 ימים/יום לפני/באותו יום/יום אחרי/מותאם) + תזמון אוטומטי |
| ✉️ תבניות מייל | עורך תבניות עם תצוגה מקדימה חיה, לכל סוג אירוע |
| 📊 דוחות | סיכומים, אירועים לפי סוג, התפלגות חודשית, אירועים קרובים |
| 🔐 משתמשים | ניהול משתמשים והרשאות (מנהל / עורך / צפייה בלבד) — מנהל בלבד |
| ⚙️ הגדרות | מיתוג, צבעים, SMTP, מודולים פעילים, החלפת סיסמה, גיבוי/שחזור |
| 📜 לוג פעילות | תיעוד כניסות, הוספה/עריכה/מחיקה, מיילים ושגיאות |

בנוסף: מצב כהה/בהיר, ממשק RTL רספונסיבי (מחשב/טאבלט/טלפון), תפריט צד דינמי.

## שליחת מיילים אמיתית

כברירת מחדל המערכת פועלת ב**מצב תצוגה מקדימה** — כפתור "שלח מייל" מציג את המייל המעוצב מבלי לשלוח.

להפעלת שליחה אמיתית: היכנסו ל**הגדרות → דוא"ל (SMTP)** והזינו שרת SMTP, משתמש, סיסמה וכתובת שולח. לאחר מכן התזכורות יישלחו אוטומטית וכפתור "שלח" ישלח מייל אמיתי.

התזמון האוטומטי רץ יומית ב-06:00 (שעון השרת): מחשב תזכורות עתידיות ושולח את אלו שהגיע זמנן.

## מבנה הפרויקט

```
server/           שרת Node.js + Express
  db.js           סכימת SQLite + נתוני דמו
  routes/         שכבת ה-API (auth, members, events, reminders, templates, settings, activity, dashboard)
  services/       לוגיקה: תאריך עברי, חישוב גיל, מייל, תזמון, לוג
public/           ה-Frontend (Vanilla JS SPA, ללא build)
  index.html      מעטפת + מסך התחברות
  css/styles.css  עיצוב RTL + מצב כהה/בהיר
  js/             api, ui, calendar, router, app + modules/
data/             family.db (נוצר אוטומטית בהרצה הראשונה)
```

## הערות ומגבלות

- **התחברות דמו** — session פשוט בזיכרון. יש תשתית הרשאות (admin/editor/viewer) הניתנת להרחבה.
- **תאריך עברי** — בצד השרת דרך `@hebcal/core`, בצד הלקוח דרך `Intl` (לוח עברי מובנה בדפדפן).
- מודולים כמו דוחות וגיבוי/שחזור מסומנים באפיון כהרחבה עתידית ואינם בליבה זו.

## איפוס נתונים

למחיקת כל הנתונים והתחלה מחדש — מחקו את תיקיית `data/` והפעילו שוב `npm start`.
